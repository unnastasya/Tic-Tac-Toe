# Tic-Tac-Toe
[Ссылка на проект](https://unnastasya.github.io/Tic-Tac-Toe/)
Проект Tic-Tac-Toe - это классическая игра Крестики-нолики на двоих игроков.
Правило одно - выигрывает тот, кто первым составит 3 символа в ряд.
